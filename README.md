## 计算公式
#### 1.受属性影响的战法基础数值部分
通过`自由指挥官`发布的攻略可知受属性影响游戏的战法，其字面上的基础数值是当属性达到80时的。但我没找到当属性低于80时该如何计算。  
于是经我自行测试得出，当属性低于80时其计算公式应该为：  
```
四舍五入((基础数值 * 0.4) + ((基础数值 * 0.6) * (属性值 / 80)))
```
